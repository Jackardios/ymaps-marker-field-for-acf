Adds a new 'Yandex Maps Marker' field to Advanced Custom Fields plugin.

== Description ==

Add a [Yandex Maps](https://yandex.ru/dev/maps/) Marker field type to Advanced Custom Fields.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `ymaps-marker-field-for-acf` folder into your `wp-content/plugins` folder
2. Activate the Yandex Maps Marker plugin via the plugins admin page
3. Create a new field via ACF and select the Yandex Maps Marker type
4. Read the description above for usage instructions
